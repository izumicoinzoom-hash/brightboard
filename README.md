# 清田自動車 工程管理アプリ

自動車鈑金・塗装工場向けの工程管理・カンバンボードアプリです。  
開発要件は `開発書.md`、**本番環境の構築手順は `本番環境構築手順書.md`** を参照してください。**Xserver へデプロイする手順**は `Xserverデプロイ手順.md` を参照してください。

## 技術スタック

- **React** (Function Components / Hooks)
- **Vite**
- **Tailwind CSS**
- **lucide-react**（アイコン）

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 主な機能

- **ログイン画面** … Googleログインボタン（UIのみ・モック）
- **カンバンボード** … 入庫予約・全作業・塗装・納車・鈑金の5種類のボード切り替え、カードのドラッグ＆ドロップ
- **代車ガントチャート** … 車両ごとの貸出予約をタイムライン表示、予約バーのドラッグで日付・車両変更
- **カード作成モーダル** … メーカー/モデル、入庫日・納車日、代車、ドット色・カード色、説明・PDF添付
- **カード詳細パネル** … カードクリックで右側にスライドイン、インライン編集
- **Googleスプレッドシート連携** … カード作成時に「カード履歴」シートへ自動で1行追加（要設定: `本番環境構築手順書.md` の「6. Googleスプレッドシート連携」参照）
